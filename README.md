# E-commerce Funnel & Churn Analysis

## 📌 Objective
Analyze user behavior to identify funnel drop-offs and customer churn patterns.

## 🛠 Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI (optional)

## 📊 Project Goals
- Perform Exploratory Data Analysis (EDA)
- Analyze conversion funnel
- Identify churned users
- Generate business insights

## 📁 Project Structure
- data/ → raw datasets
- notebooks/ → analysis notebooks
- sql/ → SQL queries
- dashboard/ → visualization files
